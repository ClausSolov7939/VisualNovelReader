Программа получает файл с текстом и метаданными для красивого вывода в
консоль. Программа выводит англо-русские символы и воспроизводит музыку.
Метаданные:
`s100` - задержка между постоянной печати символов в 100 милисекунд
`d100` - задержка между печати в 100 милисекунд
`c15` - настройка цвета шрифта с индексом 15
`w - настройка экрана(с параметрами)/ожидание клавиши
  nVNR` - настройка названия окна на 'VNR'
  x100` - настройка положения окна на 100 пикселей по X от верхнего левого угла экрана
  y100` - настройка положения окна на 100 пикселей по Y от верхнего левого угла экрана
`m - воспроизведение музыки(с параметрами)/воспроизвести один раз
  s` - остановить музыку
  l` - воспроизводить циклично
`` - перевод строки
` - стереть с экрана
По умолчанию все параметры сброшены к нулю.
P.S.: на windows 7 все функции работали, в windows 10 нет настройки экрана
